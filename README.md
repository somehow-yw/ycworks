## 云澈前端

团队基础建设文档，用于技术方案、规范等的沉淀

## 项目结构

``` js
·
├─.vuepress
│  └─public                               # 静态资源公共目录
│      └─images
├─assets
├─frontDev                                # 前端
│  ├─base
│  │  └─development-preparation-stage
│  │      └─specification
│  └─scheme
│      └─components
├─product                                  # 产品
└─teamGuide                                # 团队介绍
    └─guide


```

## 项目开发

1. 依赖安装

```js
npm install 
```

2. 本地运行文档项目

```js
npm run dev
```

3. 编译打包项目，用于生产环境部署静态网站

```js
npm run  build
```
